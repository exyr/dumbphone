### Dumbphone CLI
the smart SMS CLI for your subscription

https://www.youtube.com/watch?v=0eNNP0NJ-5Q

to start  
`source venv/bin/activate`  
`pip install -r requirements.txt`  
`python3 src/refactor`  
open localhost:8000
